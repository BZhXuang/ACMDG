# ACMDG
## Execution Guide

To run the project, you need to navigate to the source code directory and execute the main process script.

### Prerequisites
Ensure you have the following Python libraries installed:
* `numpy`
* `pandas`
* `networkx`
* `scikit-learn`
* `scipy`

### Execution Steps
1. Open your terminal or command prompt.
2. Nagitate into the `ACMDG` directory (located in the same folder as this README):
   ```bash
   cd ACMDG
3. Navigate into the `code` directory:
   ```bash
   cd code

4. Run the script using the following command in your terminal:

    ```bash
    python e_Main_And_Data_Process.py
